B. Future Unreal Conditional

1. Structure 1 (Most Commonly Used)

If ... Simple Past ..., ... would + verb ...

... would + verb ... if ... Simple Past ...

The Future Unreal Conditional is used to talk about imaginary situations in the future. It is not as commonly used as the Future Real Conditional because native speakers often leave open the possibility that anything 'might' happen.

Examples:

 • If I had some time off work, I would go to Paris.
BUT I DON'T.

 • I'm really busy next week. If I had time, I would meet you.
I do not have time to meet you.

 • Peter would help me tomorrow if he didn't have to work.
He has to work tomorrow.

2. Structure 2

If ... were + present participle ..., ... would be + present participle ...

... would be + present participle ... if ... were + present participle ...

Structure 2 of the Future Unreal Conditional is also used to talk about imaginary situations in the future.

Examples:

 • If I were going to Paris next week, I would be trying to find accommodation.
I am not going to Paris and I am not trying to find accommodation.

 • If I were not working tomorrow, I would help you. (Note that the 'were + present participle' structure is now in the 'if clause'.

 • If I had time tomorrow, I would be fishing on the lake. ( 'were + present participle' structure is in the result clause)

3. Structure 3

If ... were going to + verb ..., ... would be + present participle ...

... would be + present participle ... if ... were going to + verb ...

Structure 3 of the Future Unreal Conditional is a variation of Structure 2 which is also used to talk about imaginary situations in the future.

Examples:

 • If I were going to go to Rome, I would be trying to find accommodation.
 • If I were not going to visit my grandmother tomorrow, I would help you study.

4. Only use "If"

Only the word "if" is used with the Past Unreal Conditional because you are discussing imaginary situations. "When" cannot be used.

Examples:

 • I would buy that car tomorrow when it were cheaper. Wrong
 • I would buy that car tomorrow if it were cheaper. Correct

5. Conditional with Modal Verbs

There are some special conditional forms for modal verbs in English:

would + can = could

would + shall = should

would + may = might

The words "can," "shall" and "may" cannot be used with "would." Instead, they must be used in these special forms.

Examples:

 • If I went to UK next year, I would can learn English. Unfortunately, that's not possible. Wrong
 • If I went to UK next year, I could learn English. Unfortunately, that's not possible. Correct

The words "could," should," "might" and "ought to" include conditional, so you cannot combine them with "would."

Examples:

 • If I didn't have to work tonight, I would could go to the fitness center. Wrong
 • If I didn't have to work tonight, I could go to the fitness center. Correct

6. Future Real Conditional vs. Future Unreal Conditional

To help you understand the difference between the Future Real Conditional and the Future Unreal Conditional, compare the examples below:

Examples:

 • If you help me move tomorrow, I will buy you dinner. Future Real Conditional
I DON'T KNOW IF YOU CAN HELP ME.

 • If you helped me move tomorrow, I would buy you dinner. Future Unreal Conditional
YOU CAN'T HELP ME, OR YOU DON'T WANT TO HELP ME.
